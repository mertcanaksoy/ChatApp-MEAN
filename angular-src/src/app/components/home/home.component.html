Kullanılan Teknolojiler

node | express | mongo

socket.io

Angular

Frontend kısımda Angular 4 framework'ü kullanılmıştır

angular | angular-cli

Kurulum İçin Dikkat Edilmesi Gerekenler

  • Terminalden projenin kök dizinine gidin (cd "proje klasörünün adı")
  • .env.example dosyasını baz alarak .env dosyasını oluşturun
  • npm install komutunu yazın ve modüllerin yüklenmesini bekleyin
  • terminalden cd angular-src komutunu yazarak bu klasöre geçiş yapın
  • npm install komutunu yazın ve modüllerin yüklenmesini bekleyin
  • cd ..\ ile kök dizine geri dönün
  • npm run dev komutuyla sunucuyu çalıştırın
  • Yeni bir terminal açın
  • Terminalde cd angular-src komutuyla ilgili dizine geçin
  • npm start komutuyla uygulamayı çalıştırın
  • Tarayıcınızda http://localhost:4200 adresine gidin
  • Olası hata ihtimaline karşı terminalden projenin kök dizinine gidip npm install socket.io komutuyla socket.io'yu manuel olarak projeye dahil edin
